# VibePet 桌面宠物使用指南

欢迎使用 VibePet! 这是一只住在你桌面上的小伙伴，会卖萌、会提醒、会陪你聊天。下面是它的全部功能和使用方法。

---

## 基本操作

### 点击宠物

用鼠标左键点击宠物，它会冒出小爱心粒子，同时跟你打个招呼或说句话。

### 拖拽宠物

按住鼠标左键不放，移动鼠标超过一小段距离后，宠物就会进入奔跑状态，跟着你的鼠标走。松开后它会蹦一下落地。

### 宠物会自己动

如果你一段时间没有操作鼠标，宠物会慢慢进入发呆、思考状态，最后趴下睡觉。当你再次移动鼠标时，它会惊喜地跳起来。

### 眼睛会跟着你看

宠物的眼睛会根据你鼠标的位置左右转动，好像一直在看着你一样。

---

## 右键菜单

在宠物身上点击鼠标右键，会弹出功能菜单。以下是每个选项的说明。

### 动作演示

可以手动切换宠物的各种动画状态，看看它不同的样子：

- **待机** -- 站着不动的样子
- **打招呼** -- 挥挥手
- **跳跃** -- 原地蹦一下
- **奔跑** -- 跑步动作
- **失败** -- 垂头丧气
- **等待** -- 趴着等你回来
- **思考** -- 托着下巴想事情

### 自定义代办

打开备忘和提醒面板。你可以在里面记录文字备忘，或者设定一个倒计时提醒。

**记录备忘：** 输入文字，点击 GO 按钮或按回车键，内容会保存在列表里。点击 DONE 可以删除，点击复制图标可以把文字复制到剪贴板。

**设定提醒：** 输入包含时间的文字，比如：
- "30 分钟后提醒我站起来活动"
- "1 小时后叫我喝水"
- "15 分钟提醒我开会"

到时间后宠物会持续发出提示音并显示提醒文字，点击宠物即可关闭提醒。

**关闭面板：** 点击面板外部的空白区域即可关闭。

### 定时专注

选择一个时长，进入专注模式。专注期间：

- 宠物会保持安静，显示倒计时
- 宠物不会发呆或进入其他状态
- 倒计时结束后会发出铃声提醒

可选时长：5 分钟、15 分钟、30 分钟、45 分钟、60 分钟。

如果想提前结束，点击"退出专注"即可。

### 调节音量

弹出一个音量滑块，拖动滑块调节所有音效的音量。鼠标移开后自动收起。

### 对话模式

设置宠物的说话方式。有两种模式可以选择：

#### 基础闲聊 (Hitokoto)

默认模式，不需要任何配置。宠物会随机显示网络上的句子、名言等。轻量、无消耗。

#### 全面觉醒 (大模型)

连接 AI 大模型，让宠物真正"活"起来。选择一种性格，宠物会用对应风格跟你对话：

- **阳光开朗型** -- 积极向上，喜欢分享趣事
- **温柔体贴型** -- 安静暖心，会耐心倾听
- **高冷冰山型** -- 话少精辟，外冷内热
- **傲娇粘人型** -- 嘴上嫌弃，其实很在乎你
- **犀利毒舌型** -- 吐槽精准，但没有恶意
- **腹黑沙雕型** -- 阴阳怪气，非常搞笑
- **自定义性格** -- 自己写一段性格描述，打造专属宠物

使用全面觉醒模式需要先配置 API。

#### 接入 API

点击后弹出设置面板，填写三个信息：

1. **API 地址** -- 填入你的 API 服务地址（支持任何 OpenAI 兼容接口）
2. **API Key** -- 你的密钥
3. **模型名称** -- 要使用的模型名（默认 gpt-3.5-turbo）

填写完成后点击"保存"，会自动测试连通性。显示绿色表示成功，红色表示配置有误。

#### 关联 GitHub

填入你的 GitHub 用户名并保存。之后每次检测到你有新的代码提交，宠物会冒出一句鼓励的话。旁边有"测试"按钮可以先验证用户名是否正确。

### 设置

#### 开机自启动

开启后，电脑开机时 VibePet 会自动启动。再点一次可以关闭。

前面有圆点标记表示当前已开启。

#### 窗口置顶

控制宠物是否始终显示在所有窗口的最前面。前面有圆点标记表示当前已开启。设置会自动保存，下次启动时记住你的选择。

#### 导入宠物 (.zip)

可以导入新的宠物替换当前的宠物。点击后选择一个 `.zip` 文件即可。

宠物包的格式很简单，就是一个 zip 压缩包，里面包含：

- `pet.json` -- 宠物信息文件
- `spritesheet.webp` -- 宠物的精灵图

导入成功后宠物会立即更换。

### 退出

点击后宠物会先做一个垂头丧气的动画，3 秒后退出程序。

## 其他玩法

### 拖拽删除文件

把文件直接拖到宠物身上，文件会被送到回收站。宠物会张嘴"吃掉"文件并发出咀嚼音效。

### 气泡语录

宠物会根据当前时间自动显示不同的问候语：

- 开机时打个招呼
- 节气和节日会有特别的祝福（冬至、春节、中秋、母亲节等）
- 晚上 23:00 会提醒你早点休息

## 常见问题

**Q: 宠物挡住我看东西了怎么办？**
A: 右键菜单 > 设置 > 窗口置顶，取消勾选即可。取消后宠物会被其他窗口遮挡。

**Q: 怎么换一只宠物？**
A: 右键菜单 > 设置 > 导入宠物 (.zip)，选择新的宠物包文件。

**Q: 全面觉醒模式连不上 API？**
A: 检查 API 地址是否正确，Key 是否有效。地址会自动补全，但需要确保服务本身可以访问。

**Q: 忘记关自启动了怎么办？**
A: 打开宠物后右键 > 设置 > 开机自启动，点击取消即可。
